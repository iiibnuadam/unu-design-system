## Specification

The `Dialog` component is used for displaying modal dialogs, providing a way to present important information or capture user input.

### Anatomy

The `Dialog` component consists of:

- A title section
- An optional subtitle
- A content area for displaying custom information
- An optional footer
- A close button (optional)
